HTML Table Reference

Table Tags

Tag Name Description
<table> Table Defines a table
<th> Table Header Defines a header cell in a table
<tr> Table Row Defines a row in a table
<td> Table Data Defines a standard data cell in a table
<caption> Caption Defines table caption (must be inserted after <table>
<colgroup> Column Group Specifies a group of one or more columns in a table for formatting
<col> Column Specifies column properties for each column within a <colgroup> element
<thead> Table Head Groups the header content in seperate table
<tbody> Table Body Groups the body content in a table
<tfoot> Table Foot Groups the footer content in a table

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span
rowspan Row Span Defines how many rows a td element should span
